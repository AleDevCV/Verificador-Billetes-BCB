# 🇧🇴 Verificador de Billetes Offline (Versión BCB)

Una aplicación móvil nativa para Android diseñada para validar la autenticidad y el estado legal de los billetes bolivianos de la **Serie B**, utilizando reconocimiento óptico de caracteres (OCR) con Inteligencia Artificial, operando 100% sin conexión a internet.

## 📌 Contexto del Proyecto
A raíz del siniestro aéreo ocurrido en El Alto (marzo 2026), el Banco Central de Bolivia (BCB) inhabilitó rangos específicos de billetes de la Serie B. **esta versión de la aplicación utiliza la fragmentación matemática exacta del BCB**, garantizando precisión absoluta y evitando falsos positivos para los comerciantes.

## ✨ Características Principales
* **⚡ Validación en Tiempo Real:** Escaneo de múltiples billetes simultáneamente mediante la cámara usando `CameraX` y `Google ML Kit`.
* **📦 Análisis por Lotes:** Ingreso manual o por escáner de grandes volúmenes de números de serie, generando una tabla de resultados (Seguro / Inhabilitado) fácil de leer.
* **🔒 Privacidad y Seguridad (Offline First):** La aplicación no requiere conexión a internet. Los datos nunca salen del dispositivo. El código está protegido y firmado criptográficamente para evitar manipulaciones maliciosas.
* **📊 Transparencia:** Incluye una sección interna con las listas oficiales completas del BCB para consulta rápida del usuario.
